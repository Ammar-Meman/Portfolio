# Ammar Meman - Portfolio

A modern, highly interactive portfolio built with React, Vite, and Tailwind CSS. It showcases my journey as a Full Stack Developer, highlighting my skills, projects, certifications, and experience.

## 🚀 Features

- **Interactive UI**: Fluid animations powered by Framer Motion, featuring dynamic spotlight hover effects.
- **Premium Project Showcase**: Segmented image sliders and cursor-following interactive previews for featured work and website clones.
- **Cloudinary Integration**: Fully optimized asset delivery for PDFs and high-resolution images, keeping the local repository minimal.
- **EmailJS Contact Form**: A fully functional, serverless contact form with secure environment variables.
- **Responsive Design**: Carefully crafted for both desktop and mobile experiences using Tailwind CSS.
- **Modern Tech Stack**: Built with React (Vite), Tailwind CSS, Framer Motion, and React Icons.

## 💻 Tech Stack

- **Frontend**: React, JavaScript (ES6+), HTML5, CSS3
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons (Simple Icons, Remix Icon)
- **Asset Management**: Cloudinary
- **Email Delivery**: EmailJS
- **Build Tool**: Vite

## 🛠️ Local Development

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/Ammar-Meman/portfolio.git
   cd portfolio
   \`\`\`

2. **Install dependencies**
   \`\`\`bash
   npm install
   \`\`\`

3. **Set up Environment Variables**
   Create a \`.env\` file in the root directory and add your credentials (see \`.env.example\` for reference):
   \`\`\`env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   VITE_CLOUDINARY_CLOUD_NAME=your_cloud_name
   VITE_CLOUDINARY_API_KEY=your_api_key
   VITE_CLOUDINARY_API_SECRET=your_api_secret
   \`\`\`

4. **Run the development server**
   \`\`\`bash
   npm run dev
   \`\`\`

## 📦 Build for Production

To create an optimized production build:
\`\`\`bash
npm run build
\`\`\`
