# Ammar Meman - Personal Portfolio

![Portfolio Preview](https://res.cloudinary.com/dwusthzz5/image/upload/v1776448356/portfolio/resume/Ammar_Meman_Resume_Final.pdf) <!-- Placeholder: Replace with actual portfolio screenshot if desired -->

A modern, highly interactive personal portfolio built with **React**, **Vite**, and **Tailwind CSS**. It showcases my journey as a Full Stack Developer, highlighting my skills, projects, hackathon achievements, certifications, and experience.

**🔗 Live Demo:** [https://ammar-meman.vercel.app/](https://ammar-meman.vercel.app/)

---

## 🚀 Features

- **Interactive UI**: Fluid animations powered by Framer Motion, featuring dynamic spotlight hover effects.
- **Premium Project Showcase**: Interactive project filtering (Full Stack, Games, Hackathons, Clones) with segmented image sliders and cursor-following interactive previews.
- **Advanced SEO Setup**: Fully optimized for search engines with dynamic meta tags, a custom `sitemap.xml`, `robots.txt`, and targeted keyword indexing.
- **Cloudinary Integration**: Fully optimized asset delivery for PDFs and high-resolution screenshots directly from live deployments, keeping the local repository minimal.
- **EmailJS Contact Form**: A fully functional, serverless contact form with secure environment variables.
- **Responsive Design**: Carefully crafted for both desktop and mobile experiences using a mobile-first approach with Tailwind CSS.
- **Modern Tech Stack**: Built with React (Vite), Tailwind CSS, Framer Motion, and React Icons.

## 💻 Tech Stack

- **Frontend**: React, JavaScript (ES6+), HTML5, CSS3
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons (Si & Fa families)
- **Asset Management**: Cloudinary
- **Email Delivery**: EmailJS
- **Build Tool & Hosting**: Vite & Vercel

## 🛠️ Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ammar-Meman/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   
   VITE_CLOUDINARY_CLOUD_NAME=your_cloud_name
   VITE_CLOUDINARY_API_KEY=your_api_key
   VITE_CLOUDINARY_API_SECRET=your_api_secret
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

## 📦 Build for Production

To create an optimized production build:
```bash
npm run build
```
